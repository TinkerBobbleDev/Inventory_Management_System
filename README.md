## Quick Installation

    git clone https://github.com/ekramasif/SD_Project.git

    cd SD_Project
    
### Composer Install

    composer update
    
    composer install
    
### For Environment Variable Create
 
    cp .env.example .env
 
    
 ### For Migration table in database [Create database name as ```sdproject```]
 
    php artisan migrate
    
### Server ON ```url: http://127.0.0.1:8000/```

    php artisan serve
